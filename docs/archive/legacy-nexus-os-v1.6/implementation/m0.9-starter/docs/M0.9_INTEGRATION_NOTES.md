# M0.9 Starter Integration Notes

This scaffold is intentionally minimal. It is not a complete implementation and may need path/import adjustments for the current ALiX repo.

## Integration Points

1. Locate current `alix run` entrypoint.
2. Create a `WorkflowRun` at command start.
3. Create a single-node `TaskGraph` for the legacy run.
4. Wrap existing event emission with the canonical event envelope.
5. Wrap each built-in tool call with a permissive M0.9 `PolicyDecision`.
6. Emit minimal metrics.
7. Ensure Inspector remains compatible with legacy and canonical events.

## Do Not Add in M0.9

- Full memory kernel.
- A2A.
- Distributed workers.
- New product domains.
- Full metric catalog.
