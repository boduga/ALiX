## What is an ADR?

An Architecture Decision Record captures a significant architectural decision, its context, consequences, and alternatives considered. ADRs are the permanent record of why ALiX is built the way it is.

## Template

```markdown
# ADR-XXXX — Title

## Status
Proposed / Accepted / Superseded

## Context

## Decision

## Consequences

## Alternatives Considered

## Related Milestones
```
