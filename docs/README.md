# ALiX Documentation

## Architecture

ALiX uses a three-roadmap architecture:

- **[M-Series](roadmap/m-series-platform.md)** — Platform (runtime, coordination, context, storage, tools)
- **[P-Series](roadmap/p-series-product-intelligence.md)** — Product Intelligence (workflow, adaptation, governance, learning, executive)
- **[A-Series](roadmap/a-series-autonomous-evolution.md)** — Autonomous Evolution (self-improvement, agent generation, code evolution)
