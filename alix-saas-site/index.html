Feature Arsenal

Everything you need for safe, enterprise-grade AI coding workflows

ALiX is not just another LLM wrapper. It combines deep graph analyses, automated test pipelines, and cryptographic safety policies.

GitNexus Code Graph

Performs rigorous static-semantic analysis mapping over 17K symbol relationships. Pre-calculates exact impact and blast radius of edits upstream and downstream.

DOX Operating Contracts

Binding behavioral boundaries defined in markdown contracts (AGENTS.md). Enforces strict tabular safety limits and rules automatically. No stray changes.

Self-Repairing Test Loop

Integrates directly with Vitest/Jest. If code modifications trigger a failure, ALiX uses its tool-repair module to automatically correct the syntax or logic.

Multi-Agent Guilds

Deploys highly specialized subagents: explorer, worker, test investigator, reviewer, and docs researcher. Orchestrated dynamically by the task graph scheduler.

Policy & Approvals Daemon

Policy rules evaluate risks before execution. Highly sensitive file writes, external API access, or system alterations hold for secure interactive approvals.

TUI & Web Inspector

Features an elegant Terminal UI and a complete Web Dashboard for looking deep into execution history, memory recall, active task graphs, and audit trails.

Under the Hood

The Safe Agentic Execution Pipeline

Traditional AI agents operate blind, making raw, arbitrary changes to complex codebases. ALiX implements a highly choreographed, verification-first runtime.

1

Signal Frame & DOX Valuation

Incoming instructions are wrapped in Signal Frames. They evaluate constraints against the local AGENTS.md boundaries.

2

Impact Assessment with GitNexus

Before making edits, ALiX computes upstream callers to forecast bugs. High-risk edits block and trigger approvals.

3

Iterative Tool Repair & TDD

The worker agent patches the files while testing and verification occurs concurrently, repairing any anomalies immediately.
